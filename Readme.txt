产品使用说明

产品用途：将混乱的参考文献重新排序。

使用步骤：
将re_file.txt文件中的内容都清空。
将论文正文部分复制到re_file.txt文件中，保存并关闭。
正文不包括参考文献。
将TargetFile.txt文件中的内容清空。
将参考文献复制到TargetFile.txt文件中，保存并关闭。
点击Arrange.exe启动程序。
排序后的参考文献在compled_file.txt中查看。

注意：
论文正文中如果有例如[1-3]类型的引用，
请更换成[1][2][3]，
再启动程序。

Instructions for use of the product

Product Purpose: Reorder confusing references.

Directions of use:
Empty all re_file.txt files.
Copy the body of the paper into a re_file.txt file, save and close.
References are not included in the text.
Empty the contents of the TargetFile.txt file.
Copy the references to a TargetFile.txt file, save and close.
Click on Arrange.exe to launch the program.
Sorted references are viewed in the compled_file.txt.

Note:
If there is a citation of type [1-3] in the body of the paper,
Please replace it with [1][2][3],
Start the program again.
